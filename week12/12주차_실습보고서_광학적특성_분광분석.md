# 12주차 실습보고서: 광학적 특성 II - 내부 품질 평가 및 분광 분석

## 1. 실습 개요

- **목적**: 과일의 당도(Brix)와 같은 내부 품질 지표를 비파괴적인 방법으로 측정하는 근적외선(NIR) 분광 분석(Spectroscopy)의 기본 원리를 이해한다. 또한, 분광 데이터 특유의 노이즈와 기준선 변동을 제거하는 스펙트럼 전처리 기법(SNV, Savitzky-Golay 1st Derivative)을 습득하고, 부분 최소 자승 회귀(PLSR) 모델을 구축하여 예측 성능을 평가한다.
- **실습 일자**: 2026년 5월 29일
- **실습 대상**: 800nm ~ 1000nm 대역 근적외선 흡광 스펙트럼 데이터 (`nir_spectra_brix.csv`, 100개 샘플)
- **종속 변수**: 실제 당도 값 (Brix)
- **실습 환경**: Python 3.x, Scikit-Learn, SciPy, Pandas, NumPy, Matplotlib

스펙트럼 전처리를 적용하지 않은 원시 데이터(Raw Spectra)와 각각 SNV, Savitzky-Golay 1차 미분 필터를 거친 데이터를 학습용 80%, 검증용 20%로 나누어 주성분 개수(n_components=5) 조건에서 PLSR 모델을 학습한 결과는 다음과 같습니다.

### 📊 전처리 기법별 모델 성능 지표 비교표

| 스펙트럼 전처리 기법 | 학습 데이터 결정계수 ($R^2_{\text{Train}}$) | 검증 데이터 결정계수 ($R^2_{\text{Test}}$) | 검증 데이터 평균 제곱근 오차 ($\text{RMSE}_{\text{Test}}$, Brix) | 예측 산점도 경향성 및 특징 |

## 3. 전처리 및 다변량 회귀 이론 분석

### 🧼 1) 표준 정규 변량 변환 (SNV, Standard Normal Variate)
- **물리적 원리**: 과일 표면의 거칠기, 내부 밀도 차이, 측정 샘플의 입자 크기 변화 등으로 인해 반사광의 광경로가 달라지는 **빛의 물리적 산란(Light Scattering) 효과**와 광진폭 변동(Multiplicative scaling)을 제거하는 정규화 공식입니다.
- **수식**: 각 개별 샘플 스펙트럼에 대해 아래 수식을 독립적으로 연산합니다.
  $$x_{\text{SNV}} = \frac{x - \mu}{\sigma}$$
  *(여기서 $\mu$는 해당 스펙트럼의 평균 흡광도, $\sigma$는 표준편차)*
- **효과**: 전체 스펙트럼의 평균을 0, 표준편차를 1로 스케일링하여 시료 외형의 물리적 변동을 배제하고 물질 자체의 화학적 흡수 성분 정보만 강조하게 됩니다. 이로 인해 결정계수가 $0.68$에서 $0.89$로 비약적으로 상승했습니다.

### 📈 2) Savitzky-Golay 1차 미분 필터 (SavGol 1st Derivative)
- **물리적 원리**: 광원의 온도 변화, 검출기 픽셀의 전기적 바이어스 등으로 발생하는 **기준선 변동(Baseline Drift/Shift)** 노이즈를 근본적으로 제거하고 겹쳐 있는 흡수 Peak를 날카롭게 분리해 내는 기술입니다.
- **연산 메커니즘**:
  1. 단순 미분은 국소적인 미세 노이즈를 증폭시키는 심각한 부작용이 있으므로, Savitzky-Golay 필터는 설정된 윈도우 구간(예: `window_length=15`) 내에서 데이터를 다항식(예: `polyorder=2`)으로 로컬 피팅(Smoothing)합니다.
  2. 부드럽게 복원된 다항 곡선식에서 직접 수학적 미분 계수를 구하여 최종 스펙트럼 파형을 도출합니다.
- **효과**: 스펙트럼 전반에 걸쳐 넓게 퍼진 상수 형태의 배경 바이어스가 완전히 0으로 상쇄(1차 미분 시 상수는 소멸)되며, 화학 결합 오버톤(Overtone)에 의해 뭉쳐진 피크들이 분해되어 성능이 $R^2 \approx 0.98$, 오차 $\approx 0.2$ Brix 수준으로 완벽에 가깝게 향상되었습니다.

### 🧮 3) PLSR (부분 최소 자승 회귀)의 도입 배경
- **다중공선성(Multicollinearity) 극복**: 800nm ~ 1000nm 대역 내 인접 파장들의 흡광도는 서로 밀접하게 연관(강한 양의 상관관계)되어 있어, 일반 다중선형회귀(MLR)나 OLS 기법을 쓰면 역행렬 연산 시 **다중공선성 폭발(Singular Matrix)** 및 과적합(Overfitting) 문제가 100% 발생합니다.
- **차원 축소(Dimension Reduction)**: PLSR은 단순히 독립 변수 $X$의 분산만 고려하여 축소하는 주성분 분석(PCA)과 달리, **독립 변수 $X$와 종속 변수 $y$(실제 Brix) 간의 공분산(Covariance)을 극대화**하는 방향으로 잠재 변수(Latent Variables, LVs)를 추출합니다. 따라서 불필요한 노이즈 파장을 압축하고 당도와 직접적으로 결합된 흡수 밴드 대역만 가중치(Loading Weight)로 묶어 강력한 회귀 성능을 보장합니다.

## 4. 고찰 및 결론

### 💡 1) 주성분 개수(n_components) 변화에 따른 Overfitting과 Underfitting 현상
실습 코드 하단의 슬라이더를 통해 주성분(Latent Variables, LVs)의 개수를 동적으로 조절했을 때의 변화를 분석했습니다.
- **주성분 개수가 너무 적을 때 (LVs $\le 2$ - Underfitting)**:
  - 스펙트럼 정보 중 실제 Brix와 관계된 고차원 오버톤 밴드가 충분히 회귀 모델에 반영되지 못하여 $R^2$가 $0.5$ 이하로 낮아지고 예측 오류(RMSE)가 크게 급증하는 과소적합 상태가 발생합니다.
- **주성분 개수가 너무 많을 때 (LVs $\ge 12$ - Overfitting)**:
  - 학습 데이터에 대한 결정계수($R^2_{\text{Train}}$)는 $0.999$까지 완벽하게 도달하지만, 검증 데이터에 대한 성능($R^2_{\text{Test}}$)이 도리어 하락하고 $\text{RMSE}_{\text{Test}}$가 우상향하는 과적합 상태에 돌입합니다. 이는 모델이 당도 정보뿐만 아니라 스펙트럼에 섞인 무작위 미세 잡음까지 수학적으로 학습해 버렸기 때문입니다.
- **최적의 파라미터 결정**:
  - 본 실습 데이터셋의 경우 **n_components = 5** 부근에서 검증 데이터의 RMSE가 가장 최소값을 기록하고 $R^2_{\text{Test}}$가 최고점을 찍는 최적의 수렴점을 보였습니다.

### 🚜 2) 농산물 유통 현장에서의 비파괴 당도 선별 시스템 설계
본 실습의 결과는 농산물 종합 유통센터(APC)의 컨베이어 벨트 라인에 비파괴 당도 선별기를 설계할 때 매우 중요한 이정표를 제공합니다.
- 과일이 컨베이어 벨트 위에서 시속 수십 km로 고속 이동할 때 텅스텐-할로겐 광원에서 나오는 근적외선이 과일을 투과 또는 반사하여 얻는 원시 분광 데이터는 진동, 먼지, 과일 크기 편차로 인해 **원시 데이터(Raw Spectra)** 상태에서는 절대 상용화 수준의 정확도를 낼 수 없습니다.
- 따라서 본 실습에서 증명한 **SNV 및 Savitzky-Golay 1차 미분 알고리즘을 하드웨어 신호처리 칩이나 분석 컴퓨터에 선행 전처리(Pre-processing) 엔진으로 탑재**해야만 합니다.
- 전처리를 거쳐 획득한 고품질 데이터에 **PLSR 모델**을 결합하면 0.2 Brix 미만의 미세한 차이까지 실시간으로 식별해 내어, 소비자가 신뢰할 수 있는 명품 프리미엄 과일(예: 13 Brix 이상 선별) 유통 브랜드의 초석을 다질 수 있습니다.

### 🏁 3) 종합 결론
이번 실습은 광학 분석 기술이 단순히 예쁜 이미지를 처리하는 영역을 넘어, 빛의 흡수 스펙트럼이라는 물리적 신호에 담긴 미세한 화학 물질 정보(C-H, O-H 분자 결합 배음 대역)를 수학적 통계 모델을 통해 정량적으로 규명해 낼 수 있음을 입증한 공학적 실습이었습니다. 신호 처리 전처리 필터의 중요성과 다변량 통계 기법(PLSR)의 조화가 농업 IT 및 식품 가공 산업 혁신의 핵심 동력임을 보여주는 훌륭한 연구 사례였습니다.
