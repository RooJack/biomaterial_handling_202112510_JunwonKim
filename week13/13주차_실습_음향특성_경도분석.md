# 13주차 실습: 음향 특성 — FFT 기반 경도 분석

## 🎯 실습 목표

- **음향 비파괴 검사 원리 이해**: 충격 가진(Impact Excitation) 후 발생하는 공명 주파수와 과실 경도 간의 상관관계 파악
- **FFT(고속 푸리에 변환) 기법 습득**: 시간 도메인 타격 응답 신호를 주파수 도메인으로 변환하여 공명 피크 추출
- **경도 지수 모델링 수행**: $S = f^2 \times m^{2/3}$ 수식을 코딩 구현하여 질량 보정된 비파괴 경도 지표 산출 및 등급 분류

---

## 📊 1. 실습 데이터 개요

- **데이터 생성 방식**: 가상 감쇠 정현파(Damped Sinusoid) 기반 충격 응답 신호 (30개 샘플)
- **샘플 구성**:
  - 단단한 과일 그룹: 공명 주파수 600~900 Hz (10개)
  - 보통 과일 그룹: 공명 주파수 400~600 Hz (10개)
  - 물러진 과일 그룹: 공명 주파수 200~400 Hz (10개)
- **부가 변수**: 개별 과실 질량 150~350g 랜덤 할당
- **출력 결과**: `data/acoustic_results.csv` 자동 저장

---

## 🛠️ 2. FFT(고속 푸리에 변환) 스펙트럼 분석

- 시간 도메인 타격음 신호는 다수의 주파수 성분이 혼합된 복합 파형
- **FFT 변환** (`np.fft.rfft`):
  - 시간 도메인 배열 → 주파수 도메인의 진폭(Magnitude) 스펙트럼 변환
  - 양(Positive) 주파수 영역만 추출하여 해석 효율 제고
- **파워 스펙트럼 시각화**: X축(Hz), Y축(Magnitude) 그래프에서 공명 피크 위치 직관 확인

---

## 🔍 3. 공명 피크 자동 검출

- **검출 도구**: `scipy.signal.find_peaks` 함수 활용
- **파라미터 설정**:
  - `height`: 최대 진폭의 30% 이상인 피크만 유효 후보로 선별
  - `distance`: 인접 피크 간 최소 간격 설정으로 노이즈 피크 배제
- **지배적 공명 주파수(Dominant Frequency)**: 검출된 피크 중 가장 진폭이 큰 피크의 주파수값 추출

---

## 💻 4. 경도 지수(Stiffness Coefficient) 계산 및 등급 분류

- **수식**: $S = f^2 \times m^{2/3}$
  - $f$: 검출된 공명 주파수 (Hz)
  - $m$: 과실 질량 (kg 변환 적용)
- **질량 보정 의의**: 크기가 다른 과실 간 공정한 경도 비교를 위해 질량 가중치 적용
- **등급 분류 기준**:

| 등급 | 경도 지수 범위 | 의미 |
|------|----------------|------|
| 단단함 (Firm) | S > 200,000 | 수확 직후 또는 미숙 단계 |
| 보통 (Medium) | 80,000 < S ≤ 200,000 | 적정 숙도 구간 |
| 물러짐 (Soft) | S ≤ 80,000 | 과숙 또는 조직 연화 단계 |

---

## 📈 5. 시각화 및 인터랙티브 분석

- **4-패널 그리드 구성**:
  1. 시간 도메인 파형 비교 (단단함/보통/물러짐 대표 샘플)
  2. FFT 파워 스펙트럼 비교 + 피크 주파수 어노테이션
  3. 경도 지수 vs 공명 주파수 산점도 (등급별 색상 구분)
  4. 경도 지수 vs 과실 질량 산점도
- **인터랙티브 슬라이더**:
  - 하단 슬라이더로 질량(m) 파라미터 일괄 변경 시 경도 지수 재계산 결과 실시간 반영
  - 질량 보정이 등급 분류 결과에 미치는 영향을 직관적으로 확인 가능
- **코드 실행 방법**:
  - 터미널에서 `python step1_acoustic_fft.py` 실행 후 콘솔 분석 결과 및 플롯 팝업 확인
